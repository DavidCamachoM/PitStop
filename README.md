# 🏎️ PITSTOP 85 | Gastro-Experience

Landing page de alto rendimiento diseñada para un restaurante temático de automovilismo. Este proyecto combina una estética "racing" con una interfaz moderna, fluida y visualmente impactante.

##  El Concepto
El proyecto busca transmitir la velocidad y la precisión de los Pits de Fórmula 1. No es solo un menú digital, es una experiencia de inmersión para el comensal:
* **Diseño High-Contrast:** Uso de rojos intensos y negros profundos (estilo F1).
* **Interactividad:** Carrusel dinámico de platos con transiciones suaves en la Home.
* **Gestión de Reservas:** Sistema interactivo con validación visual mediante modales.

##  Stack Tecnológico
* **HTML5:** Estructura semántica para múltiples páginas.
* **CSS3 Custom Properties:** Uso de variables para gestión de colores, Flexbox para layouts y animaciones `@keyframes`.
* **JavaScript Vanilla:** Lógica para el carrusel de imágenes y control del modal de reservas.
* **Recursos Externos:** Google Fonts (Montserrat) y Font Awesome 6.4 para iconografía.
